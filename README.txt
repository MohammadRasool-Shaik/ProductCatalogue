## Using an IDE

You can run the system in your IDE by running the three servers in order: _RegistrationService_, _ProductsService_ and _PricingService_.

open the Eureka dashboard [http://localhost:1111](http://localhost:1111) in your browser to see that the `PRODUCTS-SERVICE` and `PRICING-SERVICE` applications have registered.  Next open the Demo Home Page [http://localhost:3333](http://localhost:3333) in and click one of the demo links.
